### sub 3:

Use o código que está em [code](code) ou copie o código abaixo:

```C++
const int led=16;

void setup() {
  pinMode(led, OUTPUT);
}

void loop() {
  digitalWrite(led, HIGH);
  delay(1000);
  digitalWrite(led, LOW);
  delay(1000);                 
}
```
